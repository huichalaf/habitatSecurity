este proyecto busca generar un sistema que ayude a los condominios a gestionar sus accesos
usando django para el backend, creamos un sistema centralizado que lleva un registro de las visitas y las personas que acceden a un condominio

para ejecutar el proyecto en modo prueba debemos hacer:
python manage.py runserver localhost:{port}

el port lo elejimos nosotros de 0 a 65000 aproximadamente, no puede ser el 80 y algunos mas
